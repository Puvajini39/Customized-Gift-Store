## 📌 Project Description
The **Customized Gift Store** is a web-based application where users can order personalized gifts online. Customers can customize products by adding names, special messages, and photos, making gifts unique for birthdays, weddings, anniversaries, and other occasions.

## 🚀 Features
- 👤 User registration and login  
- 🎨 Gift selection and customization  
- ✏️ Add names, messages, and photos to products  
- 🛒 Place and manage orders  
- 📱 Responsive and user-friendly interface  

## 🛠️ Technologies Used
- **Frontend:** HTML, CSS, Bootstrap, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL  

## 🛍️ How It Works
1. Customer signs up or logs in.  
2. Selects a gift product (mug, t-shirt, frame, keychain, etc.).  
3. Customizes the gift with name, message, and photo.  
4. Places the order for processing.

## 🎯 Objective
To provide a simple and efficient platform for customers to create personalized and meaningful gifts easily.
